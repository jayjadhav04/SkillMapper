# **App Name**: SkillMapper

## Core Features:

- Skill Input: Accept user input of current skills (e.g., Python, HTML, Communication).
- Goal Input: Accept user input defining career goals (e.g., 'Become a mobile app developer').
- Roadmap Generation: Generates a roadmap of the next 3-5 skills to learn, based on the user's current skills and career goals. An AI tool is employed to decide on incorporating pieces of information that contribute to achieving the final output.
- Resource Linking: Provides links to high-quality, free learning resources for each skill in the roadmap.
- Visual Roadmap Display: Presents the roadmap in a clear, visual format that is easy to understand.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) to represent knowledge and progression, without resorting to literal imagery like graduation caps.
- Background color: Light blue (#E3F2FD), desaturated, providing a calm learning environment.
- Accent color: Orange (#FF5722), analogous to blue, stands out for important actions or milestones.
- Body and headline font: 'Inter' sans-serif for a modern, objective, and neutral look.
- Use simple, intuitive icons to represent different skills and resources.
- Employ a clean, structured layout to make the roadmap easy to follow.
- Use subtle transitions and animations to enhance user experience.
